Hi, my name is Samuel Phillips.

I am currently a senior Computer Science student at the University of North Carolina at Charlotte. I have a concentration in Bioinformatics and a minor in Biology. Over these last few years, I have gotten education on programming languages from Java to Python to Ruby and some lab work experience in common lab techniques.

The purpose of this website was to test my existing knowledge in Python and Java and use those techniques to work on full stack development. I wanted most of the server side programming to be Python, since it’s my favorite language, so I stumbled across the web framework Django and decided to pursue the ‘Django stack.’ Frontend consists of HTML, CSS, JavaScript, Bootstrap, jQuery; Backend consists of Python, Heroku, SQLite and some Java, all with the Django framework. The mac terminal and sublime text were used to edit the code and then AWS S3 and SES to finally bring the website into deployment. All completed during the entire course of Summer 2020 during this pandemic.
